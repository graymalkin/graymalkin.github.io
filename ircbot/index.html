Gwen, My C#.NET IRC Bot

About

Gwen is a simple bot written in C#.NET to respond to $weather in IRC with the current weather conditions. It utilises Yahoo WOEIDs and their weather API to get the data

General Commands

The usage for this bot is quite simple, most modules with have a dedicated $command in their description, but I've made a few useful intrinsic commands too

Weather

The weather module, in Weather.cs, uses Yahoo WOEID APIs to get weather information from the net and puts returns that information to the sender.


Insult

Brown left Gwen in a terrible mood one morning, and she developped a hobby for creating the most vile insults. One can invoke her venomous tongue using the $insult [user] command

Admin

There are some much needed admin commands, which allow users to be added to the list of people who can use $join and $quit

Usage

The usage for this program is quite simple, most of the commands start with $ and then their name, and atributes

$join [channel name]
$leave [channel name]
$quit [reason]
$help

Weather Module

The weather module takes a location, which can be a postal code, place name, point of interest, address and so on - gets a Yahoo WOEID and then sends that to the Yahoo weather API to get the current weather conditions in that location.

$weather [location]    The default location is Canterbury, Uk

Admin

There are two commands to add and remove admins. Any admin may remove another admin - including them selves, but only graymalkin may add admins. Graymalkin cannot be removed from admins. All admin commands create a log entry.

$add_admin [username]
$remove_admin [username]
